Health Tracker

• SwiftUI
• Swift
• HealthKit
• Swift Charts

Weather Oracle

The app provides users with current weather conditions based on their location or any customized search area. Offering insights into the current, 48-hour, and 7-day forecasts, 'feels like' temperature, sunset and sunrise times, moon phases, and hourly precipitation, dew point, pressure, wind direction and speed, UV index, and visibility.

Tech Used:
• SwiftUI
• Swift
• MVVM Architecture
• Rest API
• Map Kit
• Core Location
• Animations
• Concurrency
• Network

Available on Apple Store
